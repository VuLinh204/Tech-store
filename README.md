# Tech-store

Cách 1: Tạo các image

start -> docker-compose up (8081)

dừng -> ctr+C

CáCh 2: Không tạo các image

start -> docker-compose start (8080) nhớ sửa trong docker-compose yml

dừng -> docker-compose down

npm install axios

cd Tech-store\frontend
npm start -> bật
